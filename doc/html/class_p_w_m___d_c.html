Detailed Description

Generate an alternating duty cycle signal.

The signal frequency is defined in 0.001Hz by PWM_DC::fq. The duty cycle is defined in 0.1% by PWM_DC::dc.

A sample implementation can be found in examples/signalgeneration/pwm_dc/pwm_dc.ino.
