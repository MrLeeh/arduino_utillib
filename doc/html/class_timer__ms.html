Detailed Description

A simple cyclic Timer with 1ms resolution.

This is a simple Timer with milliseconds resolution. If the Timer is started with Timer_ms::start() the internal counter is increased until it reaches the interval time set by Timer_ms::setInterval(ulong ms). Then Timer_ms::out is set to true for one cycle and the interval begins again.

The Timer_ms class uses the millis() function so the interval time can be set from 1ms to appox. 50 days.
