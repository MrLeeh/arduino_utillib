Detailed Description

A simple cyclic Timer with 1Âµs resolution.

This is a simple Timer with microseconds resolution. If the Timer is started with Timer_us::start() the internal counter will be increased until it reaches the interval time set by Timer_us::setInterval(long us). Then Timer_us::out is set to true for one cycle and the interval begins again.

The Timer_us class uses the micros() function so the interval time can be set from 1Âµs to appox. 70 minutes.
